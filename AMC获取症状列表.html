获得症状列表

接口调用请求说明

    
      https请求方式: POST
      地址：http://{{name}}/symptom/v1/list.do?access_Token=

参数说明（表1）

参数名称 是否必须 参数注释
Access_Token 是 调用接口获取的token值
AccessCode 是 访问码
Key 是 Md5( accessCode + sk )

返回值参数（表2）

正常情况下，我们会返回以下数据：

参数名称 类型 注释
Code Int 200:成功，501～510:失败
Message String 失败提示信息
AccessId String 访问码标识
SymptomList List 症状列表（详见表2-1）

SymptomList详细（表2-1）

参数名称 类型 注释
SymptomId String 症状标识
SymptomName String 症状名称
