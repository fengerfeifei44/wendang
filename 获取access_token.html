获取access_token

接口调用请求说明

    
      https请求方式: POST
      地址：http://{{name}}/oauth/get_token?

参数说明

参数名称 是否必须 参数注释
client_id 是 后台分配的id
client_secret 是 后台分配的密钥

返回值参数

正常情况下，我们会返回以下数据：

参数名称 类型 注释
code String 200:成功，非200:失败 （必会返回）
message String 提示信息 （必会返回）
access_token String access_token
expires String 有效期（秒）（43199-12小时）
scope String 目前为固定值（hm）
