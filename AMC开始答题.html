开始答题

接口调用请求说明

    
      https请求方式: POST
      地址：http://{{name}}/question/v1/start.do?access_Token=

参数说明（表1）

参数名称 是否必须 参数注释
Access_Token 是 调用接口获取的token值
AccessId 是 访问码标识
Birth 是 出生年月日：格式YYYYMMDD
Who 是 称呼1：本人，2：其他人
Sex 是 性别89：女，88：男
SymptomId 是 症状标识
Key 是 Md5(accessId + symptomId + sk )

返回值参数（表2）

正常情况下，我们会返回以下数据：

参数名称 类型 注释
Code Int 200:成功，501～510:失败
Message String 失败提示信息
SymptomId String 症状标识
Abbr String 问诊编号
Questions List 问题列表（详见表2-1）

Answers详细（表2-1）

参数名称 类型 注释
QuestionId Int 题目标识
Question String 题目内容
Detail String 题目小i内容
QuestionType Int 选项类型1:单选，2:多选，3:填空，4:简答
Answers List 答案列表（详见表2-1-1）

Questions详细（表2-1-1）

参数名称 类型 注释
AnswerIde Int 答案标识
Question String 答案内容
Detail String 答案小i内容
